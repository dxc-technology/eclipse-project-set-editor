Projekt Set Benutzervorgaben

Zum Ändern der Benutzervorgaben für die Projekt Sets verwenden sie den Menüpunkt "Fenster > Benutzervorgaben" und wählen sie auf der linken Seite "Projekt Sets". Im rechten Bereich wird folgendes angezeigt:

In der Liste werden die definierten Pfadvariablen angezeigt. Diese können bei der Definition der bevorzugten Projektverzeichnisse verwendet werden. Mit den Knöpfen rechts davon können Variablen hinzugefügt, gelöscht oder bearbeitet werden.

Mit der Option darunter kann man einstellen, ob ein Projekt vor dem Ersetzen aktualisiert werden soll. Dies kann u.a. sinnvoll sein, falls man die automatische Aktualisierung in Eclipse ausgeschaltet hat und Dateien mit externen Programmen verändert. Ansonsten kann es beim Ersetzen zu Warnmeldungen kommen, die darauf hinweisen, dass eine Datei nicht synchron zum Eclipse Arbeitsbereich ist.

Mit den Checkboxen kann das Standardverhalten des Projekt Set Editors eingestellt werden.
