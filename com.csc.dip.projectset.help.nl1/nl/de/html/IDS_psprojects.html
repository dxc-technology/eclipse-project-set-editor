Projekt Set Projekt

Konzept

Ein Projekt Set Projekt (PS Projekt) ist ein spezialisiertes Projekt mit genau einer Project Set Datei . In der Eclipse Sicht wird ein PS Projekt mit dem Symbol angezeigt.

Jedes PS Projekt hat eine Datei namens .psproject, die den projektrelativen Namen für die Haupt-Projekt-Set-Datei enthält. Wenn diese Projekt Set Datei umbenannt wird, muss der Dateiname in der .psproject Datei entsprechend geändert werden. Dieses sollte man über den Eigenschaftsdialog des Projektes auf der Seite "Projekt Set Projekt" machen.

PS Projekte geben dem Nutzer die Möglichkeit, Projekt Sets hierarchisch zu schachteln, indem PS Projekte als Einträge in anderen Projekt Sets genutzt werden. Projekt Sets können dann rekursiv geladen werden. Das bedeutet, wenn ein Projekt Set X, das ein Projekt Y enthält, geladen wird und dieses Projekt Y ein PS Projekt ist, so werden die Projekte, die in der Projekt Set Datei Y spezifiziert sind, ebenso geladen.

Projekt Set Dateien können auch ohne ein PS Projekt genutzt werden, allerdings können sie dann nicht geschachtelt werden und es werden keine Fehler-Marker für die Dateien erstellt.

Konflikte

Bei der Verwendung von PS Projekten können Konflikte auftreten. In diesem Fall ist ein Projekt mit verschiedenen Kennzeichnern eingetragen worden. Ein Konflikt entsteht zum Beispiel, wenn ein Projekt Set ein Projekt X und ein Projekt Y beinhaltet, welches ein Projekt Set Projekt ist, das das Projekt X mit einer anderen Kennzeichnung enthält.

Konflikte können nur bemerkt werden, wenn das PS Projekt, das den Konflikt enthält, mit dem im Projekt Set spezifizierten Kennzeichner geladen ist, z.B. ist in dem oberen Beispiel der Konflikt nicht sichtbar, wenn das PS Projekt Y nicht oder mit einem anderen Kennzeichner geladen ist.

Analog können Konflikte auftreten, wenn für ein Projekt verschiedene bevorzugte Projektverzeichnisse definiert werden.

Ein Projekt Set Projekt erstellen

Um ein Projekt Set Projekt zu erstellen, benutzen Sie in der Menüleiste "Datei > Neu > Andere..." und wählen Sie in der Kategorie "Projekt Set" den Typ "Projekt Set Projekt" aus.

Der Wizard für ein Projekt Set Projekt erstellt automatisch die .psproject Datei und die Projekt Set Datei mit den Projekten, die auf der Wizard Seite festgelegt wurden.

Projekt Set Fehler-Marker

Bei Änderungen im Arbeitsbereich (z.B. Ändern einer Projekt Set Datei in einem Projekt Set oder Laden eines Projektes) werden alle PS Projekte auf Fehler und Warnungen geprüft. Für jedes PS Projekt wird nur die in der Projekt Set Datei angegebene .psproject Datei auf Konflikte und Ladestatus geprüft. Andere Projekt Set Dateien im PS Projekt werden nicht geprüft.

Die Fehlermeldungen und die Warnungen sind alle in der Eclipse-Sicht "Fehler" aufgelistet und die betroffenen Resourcen werden mit einem der folgenden Symbole markiert:

  • Mindestens eines der im Projekt Set spezifizierten Projekte wurde nicht oder mit einer anderen Kennzeichnung geladen.

  • Das Projekt Set enthält Konflikte (Projekte sind in verschiedenen Kennzeichnungen spezifiziert)

  • Das PS Projekt hat einen Fehler (z.B. die Projekt Set Datei, die in der .psproject Datei spezifiziert wurde existiert nicht)

Laden von Projekten eines PS Projektes

Um die Projekte zu laden, die durch das Projekt Set des PS Projektes definiert sind, verwenden Sie die Menüpunkte "Laden/Ersetzen Projekte" oder "Laden/Ersetzen Projekte" im Kontextmenü des PS Projektes oder der darin enthaltenen Projekt Set Datei.
