Lizenz

DXC Technology Deutschland GmbH stellt alle Inhalte in diesem Plug-in zur Verfügung (nachfolgend „Inhalt“). Sofern im Folgenden nichts Anderweitiges bestimmt ist, wird Ihnen der Inhalt gemäß den Bedingungen der Common Public License Version 1.0 (nachfolgend „CPL“) bereitgestellt. Eine Kopie der CPL erhalten Sie unter http://www.eclipse.org/legal/cpl-v10.html . „Programm“ bedeutet im Sinne dieser CPL „Inhalt“.

Beiträge

Wenn der Inhalt gemäß den Bedingungen der CPL lizenziert ist, werden alle in der CPL beschriebenen, sich auf den Inhalt beziehenden Beiträge, die Sie hochladen, einsenden oder Mitarbeitern von Eclipse.org bzw. auf dem Host der Eclipse.org web site anderweitig bereitstellen, gemäß den Bedingungen der CPL behandelt und können anderen Nutzern gemäß diesen Bedingungen bereitgestellt werden.

Wenn der Inhalt gemäß den Bedingungen anderer Lizenzen bereitgestellt wird (nachfolgend „andere Lizenzen“), werden alle sich auf den Inhalt beziehenden Änderungen, Erweiterungen und/oder Codes bzw. Dokumentationen (nachfolgend „Änderungen“), die hochgeladen, eingesendet oder Mitarbeitern von Eclipse.org bzw. auf dem Host der Eclipse.org web site anderweitig bereitgestellt werden, gemäß den Bestimmungen der anderen Lizenzen behandelt und können anderen Nutzern gemäß dieser Lizenzen bereitgestellt werden. Zudem werden auch Änderungen, deren Urheber Sie sind, gemäß den Bedingungen der CPL bereitgestellt und können dann wiederum gemäß den Bedingungen der CPL anderen Nutzer bereitgestellt werden.
