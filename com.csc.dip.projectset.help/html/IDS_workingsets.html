Working Sets

A working set can be created with the content of one or more project sets .

Select one or more project set files or project set projects and select "Create working set" from the context menu. The following dialog will be opened.

If the checkbox in the dialog is selected the working set will also contain the projects specified in the sub project sets of the selected project sets. Only the currently loaded sub project sets will be used. This operation not loads any projects to the workspace.
